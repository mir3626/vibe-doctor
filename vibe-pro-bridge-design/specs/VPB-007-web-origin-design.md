# VPB-007 — Web-Origin Feature Design

## Objective

Allow Web Pro to design a new feature first and let CLI import it through the same protocol.

## Deliverables

- web-origin request creation tool
- repository/head binding
- result kind `design`
- CLI list/sync
- same importer/output contract

## DoD

A Web-created design is installed locally without download, copy, or GitHub write.
