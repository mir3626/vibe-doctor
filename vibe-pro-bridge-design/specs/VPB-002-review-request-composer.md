# VPB-002 — Review Request Composer

## Objective

Build a bounded Web Pro review request that references GitHub instead of embedding the repository.

## Deliverables

- ReviewRequest schema
- GitHub visibility checks
- optional patch attachment
- audit/design prompt templates
- output contract
- local request mirror

## DoD

The prompt contains exact refs, original design, review dimensions and required result paths;
no secret or full repository archive is included.
