## 기본 규칙 (템플릿 기본값)

- TypeScript는 `strict` 기준으로 유지한다.
- 런타임 의존성은 최소화한다.
- 가능하면 Node 표준 라이브러리를 우선 사용한다.
- 변경은 최소 범위로 한다.
- 로그는 사람이 읽기 쉽게 남긴다.
- 스크립트는 실패 원인을 명확히 출력한다.
- 문서/보고서는 짧고 결정 사항 중심으로 유지한다.
