# v1.7.6 - Dashboard and Report Visual Refresh

Date: 2026-05-05

LTS: `v1.7.3-lts` remains the stable baseline. This is a mainline UI polish release for the harness dashboard and completion report.

## Changes

- Refreshed `.vibe/harness/scripts/vibe-dashboard.mjs` and `.vibe/harness/scripts/vibe-project-report.mjs` with a restrained operational visual system.
- Replaced the prior ornate/glass-heavy brand treatment with a compact inline SVG brand mark and Geist/Geist Mono typography.
- Reduced section-as-card nesting so repeated cards remain the primary framed elements.
- Added layout-safe preview rows for empty dashboard/report data surfaces:
  - phase/sprint roadmap previews,
  - empty daily timeline previews,
  - empty attention/risk previews,
  - empty iteration/sprint/decision/verification report previews.
- Updated project-report style regressions for the new brand mark and empty preview behavior.

## Non-goals

- No dashboard API contract changes.
- No persisted dummy data writes; preview rows are render-only and marked with `data-demo="true"`.
- No new frontend dependencies or external icon packages.
- No changes to sprint acceptance policy or Generator prompts.
