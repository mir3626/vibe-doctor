# v1.6.10 - Playwright dashboard/report smoke tests

## Summary

- Added Playwright as the upstream browser test runner.
- Added `npm run test:ui` and `playwright.config.ts`.
- Added real-browser smoke coverage for:
  - dashboard state rendering and attention toasts,
  - project report sprint rendering, decision filtering, and expand/collapse controls.
- CI now installs Playwright browser support and runs `npx playwright test`.

## Sync Notes

- `playwright.config.ts` and `test/playwright/**` are harness-owned sync assets.
- `test/playwright` is excluded from the regular TypeScript project configs so downstream harness typecheck does not require Playwright to be installed.
- `package.json` sync owns `scripts.test:ui` but still treats project `devDependencies` as project-owned.
