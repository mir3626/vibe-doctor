# v1.7.5 - Context Dependency Audit Foundation

Date: 2026-05-05

LTS: `v1.7.3-lts` remains the stable baseline. This release is a mainline report-only audit foundation.

## Changes

- Added `.vibe/harness/scripts/vibe-context-audit.mjs`.
- Added `npm run vibe:context-audit`.
- The audit scans harness skills/runbooks for path dependencies and classifies dependency strength as `hard`, `soft`, or `unknown`.
- The audit reports `known`, `missing`, `ambiguous`, and `stale` counts plus context byte overhead.
- Added regression coverage in `.vibe/harness/test/context-audit.test.ts`.

## Non-goals

- No blocking preflight, phase0 seal, sprint commit, push, or tag gate.
- No Generator prompt/context changes.
- No capsule/router prompt reduction.
- No raw contradiction-rate, domain-language, grill-pass, diagnose, or architecture-rubric enforcement in this release.
