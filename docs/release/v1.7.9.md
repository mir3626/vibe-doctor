# v1.7.9 - Diff Sidecar Foundation

Date: 2026-05-08

LTS: `v1.7.3-lts` remains the stable baseline. This is a mainline sidecar dogfood foundation release.

## Changes

- Added a manual `diff-reviewer` sidecar wrapper:
  - `npm run vibe:sidecar-run -- diff-reviewer --sprint-id <sprint-id>`
  - provider auto-selection from `.vibe/config.json.orchestrator`
  - explicit `--provider claude|codex`, `--effort high|xhigh`, and `--importance critical` overrides
- Added sealed sidecar input and artifact schemas generated from Zod.
- Added provider-specific `diff-reviewer` adapters:
  - `.codex/agents/diff-reviewer.toml`
  - `.claude/agents/diff-reviewer.md`
- Added the canonical sidecar prompt at `.vibe/harness/sidecars/diff-reviewer.md`.
- Sidecar artifacts are wrapper-written and non-durable under `.vibe/sidecars/artifacts/<sprintId>/diff-reviewer.json`.
- Added sidecar process documentation and dogfood follow-up backlog in `docs/context/sidecars.md`.

## Non-goals

- No automatic `vibe-sprint-complete`, `vibe-sprint-commit`, preflight, dashboard, handoff, session-log, or `/vibe-review` integration.
- No `wiring-reviewer`, `test-auditor`, `evidence-reviewer`, command sidecars, write-capable sidecars, or parallel lane work.
- No sidecar authority over rework/block/pass decisions.
