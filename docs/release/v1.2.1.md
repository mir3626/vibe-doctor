# vibe-doctor v1.2.1 Release Notes

Released: 2026-04-16

v1.2.1 completes the Ouroboros dependency purge. The native interview (`scripts/vibe-interview.mjs`) is now the only supported interview engine, and legacy installation guidance, fallback wording, and MCP server registration have been removed from live docs and scripts.

## Upgrade path

```bash
npm run vibe:sync
```

Migration `1.2.1` removes the legacy MCP server entry from `.mcp.json` when present and bumps `harnessVersionInstalled` without downgrading newer projects.
