# v1.7.2 - Agent Prompt Template Hardening

## Changes

- Hardened `/vibe-init --mode=agent` prompt rendering so only the body between the delegation start marker and `## (Template 끝)` is treated as live prompt content.
- Normalized wrapped one-line project descriptions with Unicode whitespace before rendering the agent delegation prompt.
- Added regression coverage that renders the checked-in `.claude/templates/agent-delegation-prompt.md` for both Codex and Claude runtimes.
- Added a long Korean/Unicode one-liner fixture to protect CLI and skill paths from line wrapping drift.
- Replaced the `/vibe-review` inline `tsx -e` helper with `.vibe/harness/scripts/vibe-review-inputs.mjs`, including deterministic dependency preflight and `--install` handling.
- Made review input collection tolerate missing or malformed sprint status for explicit init/bootstrap/harness failure reviews in partial checkouts.
- Added `.claude/templates/**` to the sync manifest so downstream clones receive delegation template fixes.
