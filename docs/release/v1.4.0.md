# v1.4.0 release note draft

- Adds an optional read-only localhost dashboard at `npm run vibe:dashboard`.
- Adds the attention pipeline: Claude Code `Notification` hooks and explicit `vibe:attention` markers append urgent events that the dashboard can surface through Web Notifications.
- Adds date-sharded daily JSONL history under `.vibe/agent/daily/`, written by harness scripts through a fire-and-forget shim.
- Keeps the feature dormant by default. Enable auto-start with `.vibe/config.json.dashboard.autoStart=true`.

The final `harnessVersion` bump is intentionally deferred to the integration release sprint.
