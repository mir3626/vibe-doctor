# v1.5.11

## Summary

Scopes `/vibe-sync` post-sync typechecking to harness code instead of downstream product code.

## Changes

- Added `tsconfig.harness.json` as the syncable harness-only TypeScript project.
- `/vibe-sync` post-verify now runs `npx tsc -p tsconfig.harness.json --noEmit` when that file exists.
- Product-owned `tsconfig.json` is no longer a hybrid harness merge target.
- Updated the template `typecheck` script to use the harness tsconfig.
- Added regression coverage for post-sync typecheck command selection.

## Verification

- `npm run typecheck`
- `npm run build`
- `npm test`
