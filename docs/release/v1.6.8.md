# v1.6.8

## Summary

Adds review-time wiring drift detection so created-but-unwired harness scripts are surfaced before they become dead artifacts.

## Changes

- Added `collectWiringDriftFindings()` to `src/lib/review.ts`.
- `collectReviewInputs()` now returns `wiringDriftFindings`.
- The detector scans `scripts/vibe-*.mjs` for runtime references in package scripts, CLAUDE/AGENTS/GEMINI memory, Claude settings, and Claude/Codex skills.
- The detector also checks whether each script is present in `.vibe/sync-manifest.json`.
- `/vibe-review` skill instructions now auto-seed Blocker findings for wiring drift entries.
- Closed `gap-review-catch-wiring-drift` as covered.
