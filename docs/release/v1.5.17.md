# v1.5.17

## Summary

Hides copied template sprint state from statusline, dashboard, and project report surfaces before `/vibe-init` has initialized a downstream project.

## Changes

- Claude statusline now renders copied `vibe-doctor` template state in downstream checkouts as `idle (0/0)` with zero open risks.
- Dashboard API hides template sprint, roadmap, iteration, product, and handoff data before init.
- Project report generation hides template sprint, milestone, session, handoff, and iteration data before init.
- Added regression coverage for statusline, dashboard state, and report rendering.
