# v1.5.15

## Summary

Routes project initialization through agent skills and exposes the shared vibe-doctor skills to Codex.

## Changes

- `npm run vibe:init` now rejects direct shell execution with guidance to start from `/vibe-init` or the Codex `vibe-init` skill.
- Agent skills can still run the mechanical bootstrap with `npm run vibe:init -- --from-agent-skill`.
- Added `.codex/skills/*/SKILL.md` wrappers that delegate to the shared `.claude/skills/*/SKILL.md` runbooks.
- Updated `/vibe-init` skill instructions to use the guarded bootstrap command.
- Added regression coverage for the init guard and Claude/Codex skill parity.
