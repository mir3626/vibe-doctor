# vibe-doctor v1.3.0 Release Notes

Released: 2026-04-16

v1.3.0 adds the iteration layer that lets a completed project roadmap roll into
the next planning and delivery cycle. It also strengthens project reporting,
review regression checks, and downstream sync paths while keeping the release
backward-compatible.

## New features

- HTML project reports are generated automatically and opened in the browser
  through `scripts/vibe-project-report.mjs`.
- `/vibe-iterate` starts the next iteration and records iteration state in
  `.vibe/agent/iteration-history.json`.
- `.vibe/agent/*` init state is reset so prior dogfood and harness records are
  removed from newly initialized projects.
- `/vibe-review` now includes regression verification by comparing previous
  review issues against current coverage.
- Review scoring now uses
  `priority_score = 10·agent_friendly + 5·token_efficient + 1·user_fyi` and
  favors script-wrapper oriented fixes.

## Upgrade path

```bash
npm run vibe:sync -- --from /path/to/vibe-doctor
```

## Breaking changes

None.
